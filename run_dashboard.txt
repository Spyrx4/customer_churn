=== Cara Menjalankan Customer Churn Dashboard ===

1. Aktifkan conda environment:
   conda activate DL

2. (Opsional) Jalankan FastAPI backend untuk fitur prediksi single:
   uvicorn app:app --reload

3. Jalankan dashboard:
   streamlit run dashboard.py

=== Fitur (3 Tab dalam 1 Aplikasi) ===
Tab 1 - "Dashboard Analytics" : Visualisasi & insight data churn
Tab 2 - "Prediksi Churn"      : Form prediksi single customer (butuh FastAPI)
Tab 3 - "Batch Prediction"    : Upload file untuk prediksi massal (TIDAK perlu FastAPI)

=== URL ===
Dashboard : http://localhost:8501
FastAPI   : http://localhost:8000/docs  (jika dijalankan)
